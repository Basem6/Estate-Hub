# 🏡 DreamHomes - Real Estate Landing Page

A modern, elegant, and fully responsive real estate landing page built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **GSAP**. The project is designed to showcase luxury homes and rental properties with smooth animations, high performance, and a premium user experience.

---

## ✨ Features

- 🏠 Modern and premium real estate UI
- 📱 Fully responsive for Desktop, Tablet, and Mobile
- ⚡ High performance and optimized loading
- 🎬 Smooth scroll and section animations using GSAP
- 🖼️ Optimized images with Next.js Image component
- 🎯 Reusable and scalable component architecture
- 🔍 SEO-friendly structure
- 🎨 Clean and minimal design
- 🚀 Fast page transitions and animations
- ♿ Accessible and user-friendly interface

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/dreamhomes.git
```

Go to the project directory

```bash
cd dreamhomes
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```text
http://localhost:3000
```
